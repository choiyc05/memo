```

-> 공급망 관리 포함
G : Governance 투명성
```

2. ESG의 이해
- **지속가능정보**에 대한 글로벌 공시 규정 강화
=> **비재무정보(지속가능성 정보)** 공시 요구가 추세

- 코로나19 이후 ESG 중요성 부각

```
=> 사업 보고서로 통합
```

3. 필요성 부각
- 기업의 이해관계자(투자자,정부,사회,고객 등)의 요구 증대
```
1. ESG 정부 규제 강화
2. 투자자의 ESG 요구 증대
3. 신용평가에 ESG 반영
4. 고객의 ESG 요구 증대
=> 기업 가치 = 재무적 + 비재무적 정보
```
