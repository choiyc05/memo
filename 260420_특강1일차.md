1. ESG 이해와 국내외 동향
기업의 **지속 가능**한 성장을 위한 3대 핵심 기준
```
E : Environment 기후 변화 대응 / 자원 절약 / 탄소 배출 감소
-> 탄소 배출량? (온실 가스 배출 등도 계산)
S : Social 사회적 책임
-> 공급망 관리 포함
G : Governance 투명성
```

2. ESG의 이해
- **지속가능정보**에 대한 글로벌 공시 규정 강화
=> **비재무정보(지속가능성 정보)** 공시 요구가 추세

- 코로나19 이후 ESG 중요성 부각
- ESG 패러다임의 변화
```
기업 정보공시 ( 기존 재무재표 + 지속가능경엉 보고서 )
=> 사업 보고서로 통합
```

3. 필요성 부각
- 기업의 이해관계자(투자자,정부,사회,고객 등)의 요구 증대
```
1. ESG 정부 규제 강화
2. 투자자의 ESG 요구 증대
3. 신용평가에 ESG 반영
4. 고객의 ESG 요구 증대
=> 기업 가치 = 재무적 + 비재무적 정보
```
